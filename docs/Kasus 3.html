def langkah_metropolis(kisi_spin, T):

    # Memilih satu spin secara acak
    i = random.randint(0, N - 1)
    j = random.randint(0, N - 1)

    # Menghitung jumlah empat tetangga
    jumlah_tetangga = (
        kisi_spin[(i + 1) % N, j]
        + kisi_spin[(i - 1) % N, j]
        + kisi_spin[i, (j + 1) % N]
        + kisi_spin[i, (j - 1) % N]
    )

    # Menghitung perubahan energi lokal
    delta_E = (
        2
        * kisi_spin[i, j]
        * jumlah_tetangga
    )

    # Aturan Metropolis
    if (
        delta_E <= 0
        or random.random() < np.exp(-delta_E / T)
    ):
        kisi_spin[i, j] *= -1

    return kisi_spin

#Menjalankan minimal 200.000 langkah Monte Carlo untuk memastikan sistem telah mencapai kesetimbangan termal sebelum pengumpulan data makrostate dilakukan.
riwayat_M = []

for langkah in range(jumlah_langkah):

    # Melakukan satu percobaan flip spin
    kisi_spin = langkah_metropolis(
        kisi_spin,
        T
    )

    # Menghitung magnetisasi setiap 100 langkah
    if langkah % interval_pengukuran == 0:
        M = np.mean(kisi_spin)
        riwayat_M.append(M)

print("Magnetisasi akhir =", np.mean(kisi_spin))

Magnetisasi akhir = 0.015

# Analisis Hasil

Simulasi dilakukan pada temperatur **T = 4,0**, yaitu berada jauh di atas temperatur kritis Model Ising dua dimensi. Pada kondisi ini energi termal mendominasi sistem sehingga orientasi spin berubah secara acak.

Konfigurasi akhir memperlihatkan tidak adanya domain magnetik besar. Spin dengan orientasi ke atas maupun ke bawah tersebar secara acak di seluruh kisi sehingga tidak terdapat keteraturan jarak jauh.

Grafik magnetisasi menunjukkan bahwa nilai magnetisasi berfluktuasi di sekitar nol sepanjang simulasi. Kondisi tersebut menunjukkan bahwa jumlah spin ke atas dan spin ke bawah hampir seimbang sehingga magnetisasi spontan menghilang.

Hasil simulasi menunjukkan karakteristik **fase paramagnetik**, yaitu keadaan ketika energi termal lebih besar dibandingkan energi interaksi sehingga sistem kehilangan sifat feromagnetiknya.
