SIMULASI MODEL ISING 2 DIMENSI

Monte Carlo Metropolis Algorithm

Tentang Proyek

Website ini merupakan hasil studi kasus Mata Kuliah Fisika Komputasi mengenai simulasi Model Ising 2 Dimensi menggunakan Algoritma Monte Carlo Metropolis. Simulasi dilakukan pada tiga kondisi temperatur yang berbeda untuk mengamati perubahan magnetisasi serta fenomena transisi fasa.

Studi Kasus

Fase Feromagnetik

T = 1.0

Mengamati pembentukan domain magnetik serta magnetisasi spontan pada suhu rendah.
