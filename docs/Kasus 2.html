def langkah_metropolis(kisi_spin, T):

    # Memilih satu spin secara acak
    i = random.randint(0, N - 1)
    j = random.randint(0, N - 1)

    # Menghitung jumlah empat tetangga
    jumlah_tetangga = (
        kisi_spin[(i + 1) % N, j]
        + kisi_spin[(i - 1) % N, j]
        + kisi_spin[i, (j + 1) % N]
        + kisi_spin[i, (j - 1) % N]
    )

    # Menghitung perubahan energi lokal
    delta_E = (
        2
        * kisi_spin[i, j]
        * jumlah_tetangga
    )

    # Aturan Metropolis
    if (
        delta_E <= 0
        or random.random() < np.exp(-delta_E / T)
    ):
        kisi_spin[i, j] *= -1

    return kisi_spin

#Menjalankan minimal 200.000 langkah Monte Carlo untuk memastikan sistem telah mencapai kesetimbangan termal sebelum pengumpulan data makrostate dilakukan.
riwayat_M = []

for langkah in range(jumlah_langkah):

    # Melakukan satu percobaan flip spin
    kisi_spin = langkah_metropolis(
        kisi_spin,
        T
    )

    # Menghitung magnetisasi setiap 100 langkah
    if langkah % interval_pengukuran == 0:
        M = np.mean(kisi_spin)
        riwayat_M.append(M)

print("Magnetisasi akhir =", np.mean(kisi_spin))

Magnetisasi akhir = 0.885

# Analisis Hasil

Pada temperatur **T = 2,27**, sistem berada di sekitar temperatur kritis sehingga energi termal dan energi interaksi spin memiliki pengaruh yang hampir seimbang. Akibatnya sistem mengalami fluktuasi yang sangat besar selama proses simulasi.

Konfigurasi akhir memperlihatkan domain magnetik dengan ukuran yang beragam tanpa adanya satu domain yang benar-benar mendominasi seluruh kisi. Pola tersebut merupakan karakteristik daerah kritis yang sering dikaitkan dengan terbentuknya struktur menyerupai fraktal.

Grafik magnetisasi memperlihatkan fluktuasi yang jauh lebih besar dibandingkan kasus temperatur rendah. Nilai magnetisasi berubah-ubah selama simulasi sebagai akibat sensitivitas sistem terhadap pembalikan spin.

Fenomena tersebut menunjukkan bahwa sistem sedang mengalami **transisi fasa**, yaitu perubahan dari fase feromagnetik menuju fase paramagnetik. Hasil simulasi konsisten dengan teori mekanika statistik mengenai perilaku Model Ising di sekitar temperatur kritis.
