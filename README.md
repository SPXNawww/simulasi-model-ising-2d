## Deskripsi Proyek
Proyek ini membahas simulasi Model Ising 2 Dimensi menggunakan metode Monte Carlo dengan algoritma Metropolis. Tujuan simulasi adalah mempelajari fenomena magnetisasi spontan dan transisi fasa pada material magnetik melalui pendekatan komputasi numerik.

Simulasi dilakukan pada kisi berukuran 20 × 20 dengan kondisi batas periodik (Periodic Boundary Condition/PBC) dan konfigurasi awal acak (Hot Start).

## Studi Kasus
1. Fase Feromagnetik (T = 1.0)
2. Titik Kritis (T = 2.27)
3. Fase Paramagnetik (T = 4.0)
